# Daily MCP

> Local Notebook + LLM + Data Visualization

一个面向个人的 MCP (Model Context Protocol) 服务器，让你通过自然语言记录和管理日常生活。可嵌入 Claude Desktop 等 AI Agent，用对话的方式轻松记录财务、待办、健康和日常点滴。

## ✨ 特性

- **自然语言交互** - 通过 AI Agent 用日常对话记录生活，无需复杂操作
- **本地优先** - 数据存储在本地 (Markdown 文件/SQLite/DuckDB)，隐私安全有保障
- **SQL 查询** - 支持灵活的 SQL 查询，满足复杂数据分析需求
- **数据可视化** - 返回结构化数据，配合 Agent (如 Claude Artifacts) 生成图表
- **多维度记录** - 财务/股票、待办任务、健康数据、日常日志一站式管理

## 🎯 核心功能

### 💰 财务管理 (Finance)

记录日常收支、投资和股票交易信息。

**功能：**
- 记录收入/支出，支持分类和备注
- 股票买卖记录与持仓追踪
- 支持 SQL 查询历史数据
- 返回结构化数据，可由 Agent 生成月度收支趋势、支出分类占比等图表

**示例对话：**
```
用户：今天午餐花了 35 元，晚上买了本书 68 元
用户：买入 500 股腾讯，成本价 380
用户：查一下这个月的餐饮支出
用户：画一个最近三个月的支出趋势图
```

### ✅ 待办任务 (Todo)

管理日常任务，支持主题分组和完成情况追踪。

**功能：**
- 创建/完成/删除待办事项
- 按主题（项目）分组管理
- 对话时自动检查并提醒当日/逾期未完成任务
- 按主题查看完成率统计

**示例对话：**
```
用户：添加一个待办：周五前完成项目报告，主题是工作
用户：完成了"买牛奶"这个任务
用户：看看"健身"主题下还有哪些没完成的
用户：这周完成了多少任务？
```

### 🏥 健康记录 (Health)

追踪个人健康指标，建立长期健康档案。

**功能：**
- 记录血压、血糖、心率、体重等指标
- 运动和睡眠记录
- 返回历史数据，可由 Agent 生成趋势图表
- 支持查询异常值记录

**示例对话：**
```
用户：今天血压 125/82，心率 72
用户：昨晚睡了 7 小时，睡眠质量还行
用户：画一下最近一个月的血压变化图
```

### 📝 日常日志 (Daily Log)

自由记录生活点滴，作为其他模块的原始数据来源。

**功能：**
- 自由文本记录
- 可关联到财务、健康等其他模块
- 支持按日期/关键词检索

**示例对话：**
```
用户：今天早上跑了 5 公里，感觉不错，中午和老王吃了顿火锅花了 150
```
> Agent 侧 AI 自动解析用户意图，调用多个 MCP Tools 完成：运动记录（跑步 5km）+ 餐饮支出（150 元）

└──────────────────────────────────────────┘
